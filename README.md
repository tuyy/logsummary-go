 go v1.15.3

### logsummary-go
 로그 파일을 입력으로 받아서 summary를 출력한다.

#### summary에 포함된 정보
 startTime, endTime, total, fileSize, inputFileName, keyValueCount
 
#### 개발 진행 순서
- [ ] 파일을 입력 받아 summary 출력
- [ ] 입력된 summary 정보를 파일로 저장 및 summary 목록 노출
- [ ] 웹으로 파일을 업로드 받아 summary 출력
- [ ] 웹으로 summary 목록 노출, 삭제
